# OllieOatMealHelperBot

This bot was coded in Python by @OllieOatMeal.

## The bot has the follwing features

### Logging

* The bot logs all actions both moderation and others. It logs user's messages, joins and leaves.

### Applications

* The bot has its own application panel where staff can open/close applications as needed.
* Users who are a certain rank can create/edit new applications.
* Users wishing to apply fill out a discord form before its sent for marking.
* Once submitted the marking team are notified and can choose to accept/deny or put on hold; the applying user gets a DM in anycase.


### Blacklists

* Staff are able to add users to a ticket/application blacklist where they wont be able to open tickets / apply.
* There is a list of blacklisted users available to view on command.
* If a user is blacklisted / removed from blacklist, they are sent a notifying DM.

### Embeds

* Allows staff members to send embeds where they can fill out a Header, body and footer.
* They can send an embed to whichever channel they wish.

### Moderation

* Staff members have access to:
    mute/unmute
    kick
    ban/unban
    lock/unlock
    nickname
    warn
    warnings
    purge
    slowmode
* These are locked to different ranks.
* Staff also have the ability to DM a user through the bot.

### Utility

* This allows for the promotion/demotion of users within the staff hierarchy.
* There are also preset messages that can be sent to remind users how to get support ect.

### Reaction Roles

* Allows staff to setup reaction roles which can add/remove one or more roles.

### Music

* Allows users to play music through the bot by sending a query to find on YouTube.
* There is a full panel for controling the full music panel.
